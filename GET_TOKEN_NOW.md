## Step-by-Step (If Link Doesn't Work)

### Step 1: Go to GitHub Settings
1. Go to: **https://github.com**
2. Click your **profile picture** (top right corner)
3. Click **"Settings"**

### Step 2: Go to Developer Settings
1. Scroll down the **left sidebar**
2. Look for **"Developer settings"** (near the bottom)
3. Click it

### Step 3: Go to Personal Access Tokens
1. Click **"Personal access tokens"** in the left sidebar
2. Click **"Tokens (classic)"**

### Step 4: Create Token
1. Click the green **"Generate new token"** button
2. Click **"Generate new token (classic)"**

### Step 5: Fill Out Form
1. **Note:** Type "My Computer" or "RF Pricing Project"
2. **Expiration:** Choose "90 days" or "No expiration"
3. **Check the box:** "repo" (this gives permission to push code)
4. Scroll down and click **"Generate token"**

### Step 6: Copy Token
1. **COPY THE TOKEN IMMEDIATELY!**
2. It starts with `ghp_` and is a long string of characters
3. You'll only see it once - save it somewhere!
